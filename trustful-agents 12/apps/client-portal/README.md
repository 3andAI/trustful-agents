# Client Portal

Portal for clients to discover and verify AI agent trust credentials.

## Features

- **Agent Discovery**: Browse validated agents
- **Trust Verification**: View collateral, T&C, and claim history
- **Trust Badge**: Generate embeddable trust badges
- **Agent Card Extension**: View A2A-compatible trust info

## Setup

```bash
pnpm install
pnpm dev
```

## Tech Stack

- Next.js 14 (App Router)
- wagmi + viem
- TailwindCSS
- @trustful-agents/sdk
