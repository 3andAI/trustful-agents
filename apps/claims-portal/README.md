# Claims Portal

Portal for claimants to file and track claims against AI agents.

## Features

- **File Claims**: Submit claims with evidence and payment receipts
- **Track Claims**: Monitor claim status through lifecycle
- **Evidence Management**: Submit additional evidence
- **Deposit Management**: View and track claim deposits

## Setup

```bash
pnpm install
pnpm dev
```

## Tech Stack

- Next.js 14 (App Router)
- wagmi + viem
- TailwindCSS
- @trustful-agents/sdk
