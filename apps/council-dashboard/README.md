# Council Dashboard

Dashboard for council members to review and vote on claims.

## Features

- **Pending Claims**: View claims awaiting review
- **Evidence Review**: Access all evidence for claims
- **Voting**: Cast votes with reasoning
- **Vote History**: Track personal voting record
- **Council Stats**: View council performance metrics

## Setup

```bash
pnpm install
pnpm dev
```

## Tech Stack

- Next.js 14 (App Router)
- wagmi + viem
- TailwindCSS
- @trustful-agents/sdk
